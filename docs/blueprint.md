# **App Name**: FreelanceRate

## Core Features:

- Rate Calculation: Calculate the total cost based on the duration and hourly rate, considering weekend meeting unavailability for longer projects.
- Tiered Pricing: Implement tiered pricing based on project duration, with dynamic display of pricing tiers (Less than 7 days, 1 week to 2 weeks, 2 weeks to 1 month, 1-4 months, 4-6 months, 6-12 months).
- Hourly Rate Input & Custom Days: Allow users to select a duration tier, input custom days within that tier's range, and see the corresponding total cost. Validate custom day input against the selected tier.
- Terms Display & Benefit Highlights: Display terms of service, meeting availability details (including short notice availability for longer plans), highlight estimated working hours (based on 5-8 hours/day for short-term, 6-8 hours/day for long-term plans), and plan-specific benefits in a visually distinct card after calculation.
- Dynamic Benefit Display: Showcase attractive points tailored to the selected plan, such as UI design update frequency (unavailable for <7 days, limited for 2 weeks-1 month, large changes allowed for >1 month), meeting confirmation flexibility, and a WhatsApp share button to send the selected plan data.

## Style Guidelines:

- Primary color: Neon green (#39FF14) for a vibrant, hacker-themed aesthetic.
- Background color: Dark greenish-black (#121212) for contrast.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights.
- Body and headline font: 'Space Grotesk' sans-serif for a computerized, techy feel; pair with 'Inter' for longer body text if needed.
- Use simple, geometric icons for a minimalist design.
- A clean, single-page layout for easy navigation and quick calculations, with a visually distinct, 'premium' card displaying the calculated estimate and plan benefits.
